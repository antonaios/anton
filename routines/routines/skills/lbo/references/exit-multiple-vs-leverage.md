# Exit multiple vs leverage — the sponsor return trade-off

Loaded when Anton narrates *why* the IRR/MOIC grid moves the way it does. The
engine produces the grid; this explains the dynamics behind it.

**The three return drivers in an LBO:**

1. **EBITDA growth** — more earnings to apply the exit multiple to.
2. **De-leveraging** — cash sweep pays down debt, so a larger share of a fixed
   enterprise value accrues to equity. This is leverage *working for* the
   sponsor.
3. **Multiple expansion/compression** — exiting above (below) the entry
   multiple adds (subtracts) return. Sponsors underwrite flat-to-conservative;
   relying on expansion is a red flag.

**The trade-off:** higher entry leverage amplifies equity returns when the deal
performs (a thinner equity cheque against the same EV gain) but *compresses*
the IRR's sensitivity to the exit multiple — because more of the value walk is
fixed debt paydown and less is equity riding the multiple. So a highly-levered
deal's grid is flatter across the exit axis; a lightly-levered deal's grid
swings more with exit multiple.

**Reading the grid:** scan the exit axis (rows) at the central entry column. If
IRR barely moves across exit multiples, leverage is doing the work and the
deal is exit-multiple-robust. If it swings hard, the thesis leans on the exit —
flag that dependence in commentary. Never tune the exit multiple to hit a
hurdle ([no-llm-maths](../../../../../OS%20AI%20Vault/CLAUDE.md#no-llm-maths)).
